### Prerequisites
You do not need to install any software to run this project.

## Built with
- Swiper (The Most Modern Mobile Touch Slider)
- jQuery v3.6.0
- Bootstrap v5.1.2
- vanilla javascript

## It includes the following features
* The menu section is dynamic
* Being Responsive 
* Customized Design
